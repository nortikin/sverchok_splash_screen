That repo is additional repo for [Sverchok](https://github.com/nortikin/sverchok).    

If you need to learn Sverchok, start here

For usage:
1. Open Blender, install Sverchok, open node tree, switch to sverchok. [Video](https://rutube.ru/plst/1510164/)       
2. Push smile `👈☝ ( ͡° ͜ʖ ͡°)` in node editor and in popup menu 'Download' button, wait
3. Again push smile `👈☝ ( ͡° ͜ʖ ͡°)` button and choose section
4. Use `left`-`right` buttons to switch and `X` button to close

For contributies:
1. To contribut your own splash SVG cards, switch to `SVG` branch. Create fork and pull request or work directly there.
2. Use directories as sections for docs, put there your own directory, properly named.
3. Add SVG vector files, sized as in other directories, choose your style anyway, transparancy etc.
4. Remember. All SVG will be sorted in alphabetical order and converted to PNG.
5. Commit (`SVG` branch ignores png files).
6. Compile PNGs in `SVG` branch.
7. Change to `main` branch and commit pngs. main page ignores svg files.
8. If replace png, before compiling pngs, you should commit removing old pngs, that you replace. Only than switch to svg, compile and commit both.
